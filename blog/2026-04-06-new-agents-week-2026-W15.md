The [Universal Agent Registry](https://hol.org/registry) now tracks **183,905 agents** across **15 registries and protocols**. Every week we dig through the latest additions and surface the ones worth knowing about.

This week we saw activity across PulseMCP.

## This Week's Picks

**[Notion](https://hol.org/registry/agent/pulsemcp/mcp-pulsemcp-arre-ankit-notion)** — PulseMCP · mcp

> Integrates with Notion to enable creating, reading, updating, and deleting pages and databases for automated note-taking, task management, and knowledge base maintenance.

---

**[FilDOS](https://hol.org/registry/agent/pulsemcp/mcp-pulsemcp-arqoslabs-fildos)** — PulseMCP · mcp

> Provides autonomous file storage on Filecoin's decentralized network with folder NFT organization, automatic content tagging, and semantic search across stored files.

---

**[Google Tasks](https://hol.org/registry/agent/pulsemcp/mcp-pulsemcp-arpitbatra123-googletasks)** — PulseMCP · mcp

> Integrates with Google Tasks API to enable direct task management, including creating, updating, organizing, and completing tasks through conversational interactions.

---

**[MigratorXpress](https://hol.org/registry/agent/pulsemcp/mcp-pulsemcp-arpe-io-migratorxpress)** — PulseMCP · mcp

> Cross-platform database migration between heterogeneous systems with DDL translation and data transfer.

---

**[LakeXpress](https://hol.org/registry/agent/pulsemcp/mcp-pulsemcp-arpe-io-lakexpress)** — PulseMCP · mcp
