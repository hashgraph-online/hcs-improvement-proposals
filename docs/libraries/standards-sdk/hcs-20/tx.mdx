Note
- These builders are for direct transaction construction (e.g., with the Standards Agent Kit or custom pipelines).
- For most applications, prefer the higher‑level client (Node/Go) or browser client (wallet).

Sources
- TypeScript: https://github.com/hashgraph-online/standards-sdk/tree/main/src/hcs-20
- Go: https://github.com/hashgraph-online/standards-sdk-go/tree/main/pkg/hcs20

## Deploy

<LanguageTabs>
<TabItem value="typescript">

```ts
import { buildHcs20DeployTx } from '@hashgraphonline/standards-sdk';

const tx = buildHcs20DeployTx({
  topicId: '0.0.100',
  name: 'Loyalty Points',
  tick: 'loyal',
  max: '1000000000',
  lim: '10000',
  metadata: 'ipfs://…',
  memo: 'deploy',
});
```

</TabItem>
<TabItem value="go">

```go
import "github.com/hashgraph-online/standards-sdk-go/pkg/hcs20"

tx, err := hcs20.BuildHCS20DeployTx(hcs20.DeployTxParams{
	TopicID:  "0.0.100",
	Name:     "Loyalty Points",
	Tick:     "loyal",
	Max:      "1000000000",
	Lim:      "10000",
	Metadata: "ipfs://…",
	Memo:     "deploy",
})
if err != nil {
	log.Fatal(err)
}
```

</TabItem>
</LanguageTabs>

## Execution

<LanguageTabs>
<TabItem value="typescript">

- **Node**: `await tx.execute(client).then(r => r.getReceipt(client))`
- **Browser**: `await (await tx.freezeWithSigner(signer)).executeWithSigner(signer)` then `getReceiptWithSigner`

</TabItem>
<TabItem value="go">

```go
response, err := tx.Execute(hederaClient)
if err != nil {
	log.Fatal(err)
}
receipt, err := response.GetReceipt(hederaClient)
```

</TabItem>
</LanguageTabs>
